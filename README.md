# SQL

## Development

```sh
# lint
$> npm run lint

# format code
# ⚠⚠⚠ runs prettier-format and eslint --fix, it will modify your code. ⚠⚠⚠
$> npm run format

# fix dependencies issues
# ⚠⚠⚠ runs lerna-audit-fix, it will modify your code. ⚠⚠⚠
$> npm run audit-fix
```
