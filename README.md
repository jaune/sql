## Development

```sh
# lint ts
$> npm run lint

# format code
# ⚠⚠⚠ runs prettier-format, it will modify your code. ⚠⚠⚠
$> npm run prettier-format

# fix dependencies issues
# ⚠⚠⚠ runs lerna-audit-fix, it will modify your code. ⚠⚠⚠
$> npm run lerna-audit-fix
```
